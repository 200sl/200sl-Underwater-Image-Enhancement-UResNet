# Underwater Image Enhancement using UResNet

## Overview

This repository contains my Master's research project completed as part of the Master of Engineering Studies (Computer Systems Engineering) at the University of Auckland.

The project investigates underwater image enhancement using deep learning and computer vision techniques to improve image quality in underwater environments affected by colour distortion, scattering, low contrast, and visibility degradation.

---

## Research Areas

- Underwater Image Enhancement
- Computer Vision
- Deep Learning
- Image Processing
- UResNet
- OpenCV

---

## Objectives

- Improve underwater image visibility
- Restore colour fidelity
- Enhance contrast and sharpness
- Evaluate enhancement performance using objective metrics

---

## Evaluation Metrics

- PSNR

- UCIQE
